# Chez Nguyen Restaurant

A refreshed marketing site for Chez Nguyen Restaurant built with Next.js 14 and Tailwind CSS. The site highlights a modern Vietnamese menu, responsive design, and clear calls to action for reservations and contact.

## Getting started

```bash
npm install
npm run dev
```

> If installation is blocked in your environment, ensure access to the npm registry or install dependencies via an approved mirror.
